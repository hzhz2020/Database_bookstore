The zip file includes 6 documents: the final report, a binary version of SQL database, a SQL  CREATE document, data files, SQL queries file, SQL insert and delete file. 


In the final report ( CSE 3241 Project Final Report.pdf), we provide the database description section and the user manual section;


Bookshop.sqlite is the binary version of SQL database;


SQL CREATE.txt  contains all of the scripts needed to create the bookstore database;


DATA Files contains all the data loaded into the database containing the given data downloaded from the project website;


SQL QUERIES.txt contains the SQL queries used from final project from PART I.


SQL INSERT DELETE.txt contains the sample INSERT and DELETE statements provided in the user manual.
